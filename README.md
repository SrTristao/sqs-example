# SQS-Example

## PROJETO

### Instalação

- npm install / yarn install
- [AWS CLI](https://aws.amazon.com/pt/cli/)
- [Serverless Framework](https://serverless.com/framework/docs/providers/aws/guide/installation/)

### Run Local

- npm run offline
- yarn offline

Default Port 3000

### Deploy

- npm run {stage}
- yarn {stage}
